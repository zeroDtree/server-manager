# 账号登记信息

收集以下信息，汇总后保存为 `data/account_prepare/registration.xlsx`（表头需与 [`account_prepare/registration_columns.yaml`](../account_prepare/registration_columns.yaml) 一致）。

| 表头        | 说明                                                                                             |
| ----------- | ------------------------------------------------------------------------------------------------ |
| 邮箱        | 用于 GSAD、NetBird 登录及接收开通通知                                                            |
| linux账户名 | GPU 服务器 SSH 用户名；小写，以字母或 `_` 开头，仅含 `a-z` `0-9` `_` `-`，最长 32 字符，每人不同 |
| 真实姓名    | 显示名、邮件称呼                                                                                 |
| 学号        | 每人不同                                                                                         |
| 年级        | 如 `2024`、`2025` 级                                                                             |

**不收集：** GSAD / NetBird 密码（系统自动生成）。
